## 1. بيئة المختبر

### جهاز المهاجم

* Kali Linux VM
* المرجع:
https://www.kali.org/get-kali/#kali-virtual-machines

### جهاز الهدف (نظام التشغيل)

* Metasploitable 2
* المرجع:
https://docs.rapid7.com/metasploit/metasploitable-2/

### جهاز الهدف (تطبيق الويب)

* DVWA (Damn Vulnerable Web Application)
* المرجع:
https://github.com/digininja/DVWA

### تشغيل Docker

**▣ مكان الاستخدام**

يتم التشغيل داخل بيئة المختبر أو الطرفية الخاصة بالـ Sandbox.

**▶ نشر سريع**

```bash
docker run --rm -it -p 80:80 vulnerables/web-dvwa
```
