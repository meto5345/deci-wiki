# 16/6/2026 | Virtual Session | Practice | English

This document provides a reference for the lab environment and the core tools used during reconnaissance and initial target information gathering exercises.

---

## 1. Lab Environment

### Attacker Machine

- Kali Linux VM
- Reference: https://www.kali.org/get-kali/#kali-virtual-machines

### Target Machine (Operating System)

- Metasploitable 2
- Reference: https://docs.rapid7.com/metasploit/metasploitable-2/

### Target Machine (Web Application)

- DVWA (Damn Vulnerable Web Application)
- Reference: https://github.com/digininja/DVWA

### Docker Deployment

**▣ Usage Location**

Run inside your local lab or sandbox terminal.

**▶ Quick Deployment**

```bash
docker run --rm -it -p 80:80 vulnerables/web-dvwa
```
